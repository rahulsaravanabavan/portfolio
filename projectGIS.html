GeoConnect Prince Edward County

Through this project, I gained hands-on experience in using GIS to address real-world challenges, particularly in rural connectivity and infrastructure planning. Working with large datasets taught me the importance of data cleaning and optimization for smoother processing in ArcGIS Pro. I also learned how to analyze broadband accessibility and visualize service gaps, deepening my understanding of digital equity in rural areas.

When I encountered missing census tract data for PEC, I had to think critically and find alternative sources like the Esri Living Atlas, which strengthened my problem-solving skills. Mapping housing and population trends over time helped me see how spatial analysis can support decision-making for community development. Additionally, working with road networks and broadband expansion data reinforced how GIS can bridge the gap between technology and public services.

Digitizing bus stops and creating transit routes in Picton gave me practical insights into how GIS is used in transportation planning. Overall, this project not only refined my technical skills but also gave me a deeper appreciation for how geospatial data can drive meaningful change in communities.

Transit Network for Lindsay

In March 2024, we as a group of 4 developed a transit system application using ArcGIS Experience Builder, integrating GTFS (General Transit Feed Specification) data for real-time transit information. This involved creating table views for transit stops, adding routing capabilities, and implementing GTFS feeds to display bus schedules and routes.

While I successfully integrated core features like directions widgets and schedule displays, I faced challenges with CSV file integration, GTFS data synchronization, and trigger functionality. This project taught me the importance of standardized transit data formats and spatial data attributes, and it highlighted the complexities of web GIS application development.

I gained hands-on experience in web GIS development, GTFS data management, data relationships, and transit system visualization, all while learning how to address technical challenges through documentation and troubleshooting.

Project developed in collaboration with Helen Plesko.
