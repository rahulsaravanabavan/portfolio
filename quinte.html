Historical Air Photo Digitization

Project Overview

At Quinte Conservation, I led the digitization of historical aerial photographs to support long-term environmental planning and web mapping. This involved georeferencing over 2,300+ scanned images, calculating photo scales, automating spatial feature generation, and linking aerial imagery to map tiles through scripting. The outcome is a spatially accurate, searchable, and web-accessible archive of historical flight lines and imagery.

Technical Process

Manual Attribute Updates

Automated repetitive field updates (e.g., PhotoName) using Python scripts to match image files with spatial features, significantly saving time.

Point Generation Along Lines

Streamlined creation of evenly spaced photo points using ArcPy's GeneratePointsAlongLines and MakeFeatureLayer, replacing manual digitizing.

Image Format for Web Display

Converted TIFF images to JPEG using GDAL in Python to ensure compatibility with browser-based viewers and internal web mapping platforms.

Duplicate Flight Line Names

Introduced subtypes based on year to differentiate identically named flight lines across multiple historical datasets, improving database integrity.

Interactive Photo Access

Developed dynamic HTML hyperlink generation scripts to display aerial photo previews within interactive tile-based web maps.

Results & Impact

This digitization project directly supported the Planning and Regulation Department by improving access to historical aerial imagery critical for permit application reviews and environmental assessments. By automating the georeferencing and indexing process, the turnaround time for retrieving historical context on a site was dramatically reduced.

Staff can now interactively explore flight lines, view linked images, and reference accurate historical land use with confidence. The improved spatial accuracy and structured database also enhanced consistency in decision-making, especially for reviewing applications in sensitive areas like wetlands or flood zones, ultimately contributing to more efficient, transparent, and well-informed permitting processes.

Wetland Assessment Tool – Planning & Regulations Support

Project Overview

To support the Planning and Regulation Department in reviewing flagged wetland polygons for permit application assessments, I developed a custom web application using ArcGIS Experience Builder. This tool allows staff to interactively visualize, evaluate, and update wetland data in a structured and user-friendly environment.

Key Features & Implementation

Main Map & Inset Map Views

Displayed wetland polygons overlaid with contours, with an inset map showing only intersecting contour lines for precise visual analysis.

Smart Attribute Table Navigation

Selecting a record in the table automatically zooms the map to the associated wetland feature, streamlining visual review.

Review Workflow & Edit Tab

Users can classify each wetland polygon using the "Review Status" field—choosing whether to retain or delete based on visual and contextual assessment.

Smart Forms via ArcGIS Online

Integrated ArcGIS Online Smart Forms with domain-driven fields to ensure controlled and consistent data entry to maintain data integrity.

Automated Attribute Rules

Developed an Arcade expression in smart form to auto-update the "Date Reviewed" field each time a polygon is edited, ensuring accurate recordkeeping.

Public Data Collection

Created editable layers for public engagement or field validation where permitted, using controlled schema and secure submission paths.

Database Management & Data Cleaning – Hunt Block Program

Project Overview

I developed a comprehensive database management system for Quinte Conservation's Hunt Block program using Microsoft Access. This project involved migrating legacy spreadsheet data into a structured relational database, implementing robust data validation, and creating automated workflows for customer management and hunt block assignments.

Database Design & ETL Process

Schema Design & Data Import

Created structured schema in Access with proper relationships, combo box domains for data integrity, and automated the import process from legacy spreadsheets into normalized tables.

Duplicate Detection & Cleanup

Used SQL statements with DISTINCT functions to identify and eliminate duplicate records, ensuring clean data migration from the original Hunt Block spreadsheet system.

Data Validation Implementation

Implemented comprehensive validation rules including email format validation using pattern matching and null value handling for critical fields like names and contact information.

Quality Assurance Queries

Developed complex SQL queries to identify missing hunt blocks, invalid contact information, and data inconsistencies across related tables for ongoing quality control.

SQL Scripts & Data Cleaning Examples

Email Validation Query

Implemented pattern matching validation: Is Null Or ((ALike "%_@_%._%") And (Not ALike "%[ ,;]%")) to ensure proper email format and prevent invalid entries.

Comprehensive Data Quality Check

Created multi-table validation queries checking hunt block existence, name completeness, postal code length, phone number format, and email validity across all customer records.

ETL Data Integration

Developed INSERT statements with INNER JOINs to populate the customer table with cleaned data from multiple sources, ensuring referential integrity throughout the migration process.

Hunt Block Assignment Logic

Created SQL queries to identify available hunt blocks and manage customer assignments, preventing conflicts and ensuring optimal resource allocation.

Advanced Form Development & User Interface

Beyond data cleaning, I developed sophisticated Access forms with VBA programming to streamline user workflows. The customer management interface features intelligent search functionality, record navigation dialogs, and automated hunt block assignment with real-time availability checking.

Key VBA implementations include error handling for data entry operations, dynamic combo box population based on available hunt blocks, confirmation dialogs for critical operations like customer detachment, and automated record position tracking. The system also features comprehensive search capabilities across multiple fields with filter management and result navigation.

95% Data Quality Improvement

The database system now provides complete audit trails for all customer interactions, maintains data integrity through validation rules and relationship constraints, and offers staff an intuitive interface for managing hunt block assignments. This solution transformed a manual, error-prone spreadsheet system into a robust, automated database that supports Conservation's operational requirements while ensuring data accuracy and regulatory compliance.

Results & Workflow Improvement

The Experience Builder application significantly improved the Planning and Regulation Department's wetland review workflow by introducing a centralized, interactive environment for QA/QC, attribute editing, and spatial analysis. A custom QA/QC model was designed in ArcGIS Pro to pre-flag wetland polygons based on size and proximity to existing wetlands.

Using the QA/QC model as a first pass ensured reviewers focused only on potential data anomalies, drastically reducing visual clutter and review time. Edits to the "Review Status" field were controlled via domain values in smart forms, while an Arcade expression automatically stamped the "Date Reviewed" date, enabling a reliable audit trail for future verification.
